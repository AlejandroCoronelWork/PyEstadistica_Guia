[MEDIA]
DEFINICIÓN: Valor representativo de un conjunto de datos (Promedio).
USO: Para encontrar el centro matemático. Sensible a valores extremos.
FÓRMULA: Suma de datos / Cantidad total.

[MEDIANA]
DEFINICIÓN: Valor central de los datos cuando están ordenados.
CLAVE: Si hay dos valores centrales, se promedian.

[MODA]
DEFINICIÓN: El valor que más se repite en el grupo.
CLAVE: Puede haber más de una moda (Bimodal) o ninguna (Amodal).

[COMBINACIONES]
CONCEPTO: Agrupar elementos donde el ORDEN NO IMPORTA.
CLAVE: "Manzana y Pera" es lo mismo que "Pera y Manzana".
FÓRMULA: nCr = n! / (r! * (n-r)!)

[PERMUTACIONES]
CONCEPTO: Ordenar elementos donde el ORDEN SÍ IMPORTA.
CLAVE: "1ro Juan, 2do Ana" NO es lo mismo que "1ro Ana, 2do Juan".
FÓRMULA: nPr = n! / (n-r)!

[EXPERIMENTO]
DEFINICIÓN: Proceso con resultado incierto (azar).
EJEMPLO: Lanzar un dado.
ESPACIO MUESTRAL (S): {1, 2, 3, 4, 5, 6}

[PROBABILIDAD]
DEFINICIÓN: Posibilidad de que ocurra un evento (0 a 1).
SIMPLE: P(A) = Favorables / Totales.
COMPUESTA: P(A y B) = P(A) * P(B) (Independientes).
